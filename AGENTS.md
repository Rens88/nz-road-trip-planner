# App Aims

- Generate an interactive app that helps visualize planned New Zealand road trip itineraries.
- Show planned stops clearly on a high-quality map with hover and popup details.
- Draw a reasonably accurate road-following route between intended stops.
- Support multiple itinerary versions so different route ideas can be compared quickly.
- Provide both a map view and a timeline view for the same itinerary data.
- Represent each stop with basic planning information: location, date range, notes, and possible meetups.
- Make the itinerary data easy to edit without changing the app code.
- Keep the generated output easy to share as a standalone HTML file that can be opened in a browser.
- Preserve graceful fallback behavior when external routing services or map layers are unavailable.
- Leave room to add trip photos later, such as stop-level images, route highlights, or a photo-rich travel summary view.
- Keep the app suitable for planning: clear, practical, fast to scan, and focused on comparing route options.
